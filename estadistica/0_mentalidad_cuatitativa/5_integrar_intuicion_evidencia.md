# Integrar Intuición y Evidencia

### **Objetivos específicos:**

1. Que el lector aprenda el ciclo: intuición → medición → verificación → calibración.

2. Que comprenda qué puede hacer la evidencia y qué no puede hacer la intuición.

3. Que entienda cómo se forma una intuición cuantitativa mediante exposición + verificación.

4. Que vea ejemplos cotidianos de intuición calibrada (clima, energía, cuerpo, cocina).

5. Que internalice que un cuantitativo auténtico combina mente humana + evidencia objetiva.
